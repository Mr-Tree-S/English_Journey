# English_Journey

## Goal

- IELTS 6.5 in ~3 months
- Vocabulary: 6,000–7,000
- Reading: complete 1 passage within 30–35 mins
- Writing: fewer grammar mistakes, clear logic
- Listening & Speaking: daily practice, reduce hesitation
